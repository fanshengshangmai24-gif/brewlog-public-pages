# BrewLog Support

Thank you for using BrewLog.

For questions, bug reports, or support requests, please open an issue on this repository.
Creating an issue requires a GitHub account. If you are not signed in, GitHub will show its sign-in or account creation screen first.

https://github.com/fanshengshangmai24-gif/brewlog-public-pages/issues/new

Related pages:

- Privacy Policy: https://github.com/fanshengshangmai24-gif/brewlog-public-pages/blob/main/PRIVACY.md
- Terms of Use: https://github.com/fanshengshangmai24-gif/brewlog-public-pages/blob/main/TERMS.md

When reporting a problem, please include:

- App version
- iOS version
- Device model
- What you were trying to do
- What happened instead

Please do not include sensitive personal information in public issues.
