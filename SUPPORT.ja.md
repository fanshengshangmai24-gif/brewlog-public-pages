# BrewLog サポート

BrewLog をご利用いただきありがとうございます。

ご質問、不具合のご報告、サポートのご依頼は、このリポジトリで Issue を作成してお知らせください。
Issue の作成には GitHub アカウントが必要です。サインインしていない場合は、GitHub のサインインまたはアカウント作成画面が先に表示されます。

https://github.com/fanshengshangmai24-gif/brewlog-public-pages/issues/new

関連ページ:

- プライバシーポリシー: https://github.com/fanshengshangmai24-gif/brewlog-public-pages/blob/main/PRIVACY.ja.md
- 利用規約: https://github.com/fanshengshangmai24-gif/brewlog-public-pages/blob/main/TERMS.ja.md

不具合をご報告いただく際は、以下を含めていただけると助かります。

- アプリのバージョン
- iOS のバージョン
- デバイスのモデル
- 何をしようとしていたか
- 実際に何が起きたか

公開される Issue には、機微な個人情報を含めないようにご注意ください。
